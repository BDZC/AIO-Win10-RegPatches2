 Dieses Archiv enthält den vielseitigen Archivmanager WinRAR.

   Funktionen und Eigenschaften von WinRAR:

 * WinRAR arbeitet mit einem speziellen Kompressionsalgorithmus.
   Damit ist eine hohe Kompressionsrate bei ausführbaren Dateien,
   Objekt-Bibliotheken, großen Textdateien, usw. möglich.

 * Das RAR-Format kann mit beliebig großen Dateien umgehen. Die
   Größe von RAR-Archiven ist nicht beschränkt, in einem RAR-Archiv
   können sich beliebig viele Dateien befinden. Einschränkungen
   ergeben sich nur aus dem verwendeten Dateisystem, dem 
   verwendeten Betriebssystem oder dem verfügbaren Speicher.

 * WinRAR enthält eine vollständige Unterstützung für RAR- und
   ZIP 2.0-Archive und ist in der Lage, folgende Archive zu entpacken:
   7Z, ACE, ARJ, BZ2, CAB, GZ, ISO, JAR, LZH, TAR, UUE, XZ und Z.

 * WinRAR lässt sich über eine interaktive grafische Windows-
   Benutzeroberfläche und über die Kommandozeile bedienen.

 * Mit WinRAR können "solide" Archive erstellt werden, wodurch die
   Kompressionsrate um 10% - 50% im Vergleich zu herkömmlichen
   Methoden gesteigert werden kann, besonders wenn eine große
   Anzahl kleiner Dateien gepackt wird.

 * WinRAR erlaubt das Erstellen von selbstentpackenden Archiven.
   Diese so genannten SFX-Archive lassen sich an eigene Bedürfnisse
   anpassen.

 * Mit WinRAR können mehrteilige Archive erstellt werden, die
   selbstentpackend sind.

 * RAR-Archive lassen sich wahlweise durch das Hinzufügen von
   Wiederherstellungsdaten vor Beschädigungen schützen.

 * WinRAR bietet eine Reihe von Servicefunktionen, wie zum Beispiel
   Verschlüsselung von Dateien, Hinzufügen von Archivkommentaren
   und das Speichern und Wiederherstellen von NTFS-Dateisicherheits-
   daten.
